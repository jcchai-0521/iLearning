## 相关命令

```
1.查询Linux版本信息：uname -a
2.创建mysql用户：useradd mysql
3.设置mysql用户密码：passwd mysql 密码设置为（cWX467306）
4.Linux创建安装包相关路径：mkdir soft
5.从window下上传源码包到Linux上： pscp .\mysql-5.7.31.tar.gz root@192.168.237.128:/usr/soft

```
